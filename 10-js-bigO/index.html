We primarily refer to time complexity using the following notations (in order of most to least optimal/efficient):

  • Constant => O(1)
  • Logarithmic => O(logN)
  • Linear => O(N)
  • O(N logN) falls between Linear and Quadratic
  • Quadratic => O(N^2)
